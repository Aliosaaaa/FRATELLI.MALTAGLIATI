# Castagne Libere — Fratelli Maltagliati

Sito statico vetrina del progetto di recupero del castagneto.

- 5 pagine HTML statiche, Tailwind via CDN
- Foto reali ottimizzate in `assets/img/`
- Embed video TVL (YouTube)
- Pubblicato su GitHub Pages

## Struttura

```
index.html         Homepage
chi-siamo.html     Chi Siamo
il-metodo.html     Il Metodo
il-progetto.html   Il Progetto / Adotta un Castagno
prodotti.html      I Nostri Prodotti
assets/img/        Foto ottimizzate
```

## Sviluppo locale

```
python3 -m http.server 8000
# poi apri http://localhost:8000
```
